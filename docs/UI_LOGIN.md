### タイプライター（参照実装のロジックを踏襲）

- 打鍵 150ms/字、削除 75ms/字、フレーズ完了後 2000ms 待機、削除完了後 500ms 待機、配列をループ
- カーソル: 幅 4px・高さ 2.5rem・`blink 1s step-end infinite`
- フレーズ（本システム向けに Fable が改稿。この 8 本をそのまま使う）:
  1. `本荘キャンパスの「知りたい」に、ぜんぶ答える`
  2. `聞いてみよう。キャンパスのこと、ぜんぶ`
  3. `学科のこと、施設のこと、入試のこと`
  4. `あなたの進路選びに、たしかな情報を`
  5. `オープンキャンパス2026へ、ようこそ`
  6. `分からないことは、その場で解決`
  7. `秋田県立大学のいまを、案内する`
  8. `ここからの一歩を、いっしょに`

## 2. アップデート（再現に対する改良点 — これのみ許可）

1. **カーソルにブランドグラデーション**（#FF8A65→#FFEB3B→#69F0AE）を適用してよい（単色でも可）
2. 黒シートは初回マウント時に下から 0.4s でスライドイン
3. `prefers-reduced-motion`: タイプライターを止め、フレーズ 1 を静的表示。カーソル点滅も停止
4. IME 変換中 Enter で送信しない（v0.1 から継続）
5. バリデーション（1〜20文字・前後空白禁止）と 409/404 エラーメッセージは v0.1 から継続

## 3. フォーム仕様

- ログインモード: ニックネーム入力のみ + CTA「ログイン」(bg-zinc-800)
- 登録モード: ニックネーム + 属性チップ（高校生/保護者/その他、黒シート上なので zinc-800 ベースの選択 UI に調整可）+ CTA「同意して登録」→ 本システムでは「はじめる」(bg-white text-black)
- 切替ボタン: 「新規登録」/「ログインはこちら」(bg-zinc-800)
- ローディング中は参照実装同様の回転スピナー + 「処理中...」
