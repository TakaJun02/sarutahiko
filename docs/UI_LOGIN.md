# ログイン画面・認証仕様

- 版: v0.1（2026-07-11, Fable 起草）
- 参考: guidanceLLM2 の `LoginView.vue`（ユーザー名のみの register/login + ガラスパネル）を出発点に、
  ChatGPT / Gemini アプリ水準のスタイリッシュな UI に刷新する。

## 1. 認証方式（軽量ログイン）

オープンキャンパス来場者が対象のため、**パスワードなしのニックネーム登録制**とする
（個人情報の最小化。本名を要求しない）。

- **登録**: ニックネーム（1〜20 文字）+ 属性（`highschool` 高校生 / `parent` 保護者 / `other` その他）
- **ログイン**: ニックネームのみ
- バックエンドはユーザー作成/ログイン時に **opaque なセッショントークン**（UUID4 以上のエントロピー）を発行。
  フロントは localStorage に保存し、以後 `Authorization: Bearer <token>` を付与する。
- 会話スレッドはユーザーに紐づく。トークン失効（サーバ再起動等）時は自動でログイン画面へ戻す。
- 属性は回答トーンの調整（システムプロンプトに反映）と将来の分析用。

### API（ARCHITECTURE.md の API 一覧にも反映済み）

```
POST /api/auth/register  {"name": "...", "role": "highschool|parent|other"}
  -> 201 {"token": "...", "user": {"id": "...", "name": "...", "role": "..."}} / 409 名前重複
POST /api/auth/login     {"name": "..."}
  -> 200 {"token": "...", "user": {...}} / 404 未登録
GET  /api/auth/me        (Bearer) -> 200 {"user": {...}} / 401
```

## 2. ログイン画面 UI（ChatGPT / Gemini 風）

### トーン
- **ダーク基調・ミニマル・上品なグラデーション**。チャット画面（ダーク）と地続きの世界観。
- 装飾過多にしない。「AI プロダクトの入り口」としての静かな高級感を出す。

### レイアウト（モバイル縦画面ファースト）
1. **背景**: 単色ダーク（例 `#0f1115` 系）の上に、ぼかした**オーロラ風グラデーションブロブ**を 2〜3 個、
   ごくゆっくり（20s+）漂わせる。色はブランド 3 色（#FF8A65 / #FFEB3B / #69F0AE）を低彩度・低不透明度で使用。
   `prefers-reduced-motion` 時は静止画にする。
2. **中央カード**: 半透明ガラスパネル（`backdrop-blur`、細い 1px ボーダー、角丸 2xl）。内容:
   - 上部に `app-icon.png`（円形、控えめなグロー。ローディング演出と同一アイコンでブランド統一）
   - 挨拶見出し（Gemini 風の 2 行）: 「こんにちは。」/「秋田県立大学 本荘キャンパス案内 AI です」
     — 見出しにはブランド 3 色の**テキストグラデーション**を一箇所だけ使う
   - ニックネーム入力（1 フィールド。フォーカス時にグラデーションのリング）
   - 属性選択は**チップ型セグメント**（高校生 / 保護者 / その他）— 登録時のみ表示
   - CTA ボタン「はじめる」（登録）/「つづける」（ログイン）。ホバー/タップで軽い発光
   - 下部に register/login 切替リンク
3. **マイクロインタラクション**: カードは初回マウントでフェード+スライドイン（0.4s 前後）。
   エラーはカードを揺らさず、入力欄下に赤系テキストでインライン表示。
4. **ログイン成功時**: カードがフェードアウトしてチャット画面へ遷移（0.3s 程度、遷移中に白フラッシュを出さない）。

### 実装条件
- Tailwind + 素の CSS アニメーションで実装（外部アニメーションライブラリは Fable 承認制）。
- ダークのみで可（ライトテーマ対応はスコープ外）。
- バリデーション: 空文字・21 文字以上・前後空白は不可。IME 変換中の Enter で送信しない
  （`keydown.enter` は `isComposing` を確認）。

- [ ] 未認証で `/app/chat` に直アクセスするとログインへリダイレクト
- [ ] 401 受信時に自動でログイン画面へ戻る

- [ ] `prefers-reduced-motion` でアニメーション停止
- [ ] モバイル縦（±390px 幅）で崩れない
