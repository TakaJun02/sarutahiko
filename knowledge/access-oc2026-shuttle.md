このページは、秋田県立大学 システム科学技術学部 オープンキャンパス2026当日の無料送迎バス、シャトル、臨時バス、秋田駅・羽後本荘駅から本荘キャンパスへの行き方を確認するための公式案内である。来場者が「羽後本荘駅からキャンパスまでの行き方」「送迎バスはあるか」「秋田駅から行けるか」と質問したときに使う。

## 無料送迎バスの案内

システム科学技術学部 オープンキャンパス2026のお知らせには、「当日、秋田駅東口・羽後本荘駅から本荘キャンパスまでの送迎バスを運行します」と書かれている。予約が必要な便については、事前申込フォームから申し込む。

| 便 | 経路 | 予約 |
|---|---|---|
| 1 | 秋田駅東口5番乗り場 8:00 → 羽後本荘駅 9:10 → 本荘キャンパス 9:20 | 要予約 |
| 2 | 羽後本荘駅 10:05 → 本荘キャンパス 10:15 |  |
| 3 | 本荘キャンパス 12:45 → 羽後本荘駅 12:55 |  |
| 4 | 羽後本荘駅 13:05 → 本荘キャンパス 13:15 |  |
| 5 | 本荘キャンパス 15:00 → 羽後本荘駅 15:10 → 秋田駅東口5番乗り場 16:15 | 要予約 |

## 秋田駅東口の乗り場

無料送迎バスの第1便と第5便は、秋田駅東口5番乗り場が案内されている。

## 予約が必要な便

予約が必要と書かれている便は、第1便「秋田駅東口5番乗り場 8:00 → 羽後本荘駅 9:10 → 本荘キャンパス 9:20」と、第5便「本荘キャンパス 15:00 → 羽後本荘駅 15:10 → 秋田駅東口5番乗り場 16:15」である。予約は事前申込フォームから申し込む。

## 到着後の学内の流れ

送迎バスで本荘キャンパスに到着した後の学内の曲がる方向や徒歩分数は、この公式ページ本文には書かれていない。OC2026タイムテーブルでは9:00受付開始、Kゾーン（共通施設棟）での当日案内や学科説明会などが掲載されているため、到着後は当日の案内表示やスタッフの案内に従う。
