このページは、秋田県立大学 本荘キャンパスに関する「学生寮 清新寮」の情報である。オープンキャンパス2026来場者が学生生活を確認するためのページである。

## 注意事項

秋田県立大学が管理・運営する学生寮は「清新寮」のみ。所在地は南秋田郡大潟村字南2丁目2。秋田市内・由利本荘市内には、本学が運営する学生寮・アパートはない。

## 概要

清新寮は大潟キャンパス敷地内にある。約75,000平方メートルの敷地に、鉄骨鉄筋コンクリート造り地上7階建ての建物と、200台収容の駐車場などを備える。居室は全館個室。

男子は2階から4階の120室、女子は5階から7階の120室に分かれている。異性階および居室への立ち入りは禁止されている。

## 各キャンパスへの所要時間

以下は「各キャンパスへの所要時間」に関する一覧である。

| 経路 | 車での所要時間 |
|---|---:|
| 大潟キャンパスから秋田キャンパス（生物資源科学部） | 約40分 |
| 大潟キャンパスから本荘キャンパス（システム科学技術学部） | 約110分 |
| 大潟キャンパスから木材高度加工研究所 | 約30分 |

## 入寮できる方

次のいずれかに該当する方が入寮できる。

- 本学に在籍する正規生。
- 本学に正規生として入学する予定で、入学手続が完了した方。
- その他、管理運営責任者が特に認めた方。

募集時に優先される条件は次のいずれか。

- アグリビジネス学科3、4年生、または主に大潟キャンパスで教育を受ける大学院生。
- 出身地が県外の方。
- 自宅から主として活動するキャンパスまでの片道の路程が60km以上の方。
- 自宅から主として活動するキャンパスまでの片道の通学時間が1時間30分以上の方。
- その他、管理運営責任者が入寮を必要と認めた方。

## 入寮手続と時期

入寮募集時に、入寮承認願を提出して申し込む。申込状況によっては、優先基準の適用および抽選により選考し、入寮を許可する。

新入生は入学年度の4月以降に入寮できる。在学生は入寮許可後、所定の手続により入寮日を決定する。

## 寮費

以下は「寮費」に関する一覧である。

| 項目 | 内容 |
|---|---|
| 月額 | 50,000円 |
| 月額に含まれるもの | 食費、光熱水費 |
| 学生自治会費 | 年額3,000円 |
| 納入期日 | 毎月27日、口座振替 |

学生自治会費は入寮後、寮自治会で徴収される。

## 居室

各室にベッド台、机、イス、内線専用電話、物干し竿、冷暖房設備がある。クローゼットには下部の引き出しと上部の荷物収納棚がある。窓のカーテンとテレビアンテナ端子もある。インターネットは各自契約が必要だが、LAN差込口はある。

電気製品等は原則持込自由。ただし、大型のものや火気を伴うものは持ち込めない。

## 食事

食堂があり、平日は朝夕の2食、土日祝日は朝昼夕の3食が提供される。お盆休みおよび閉寮期間中は除く。

| 食事 | 営業時間 |
|---|---|
| 朝食 | 7時00分から8時30分 |
| 昼食 | 11時45分から13時30分 |
| 夕食 | 17時00分から21時00分 |

各階に湯沸室があり、ガスコンロや電子レンジを備える。軽食やお茶などの簡易な調理ができる。ガスコンロ等の使用料はかからないが、調味料などは各自負担。

## 浴室・洗濯

1階に大浴場と小浴場があり、毎週男女が入れ替わる。入浴時間は17時00分から24時00分。浴室とは別に1階にシャワー室が2部屋あり、予約により利用できる。

各階2カ所に洗面台、トイレ、洗濯機、乾燥機、乾燥室がある。洗濯機は全自動で使用料はかからない。洗剤等は各自持ち込み。乾燥室に洗濯物などを干すことができ、暖房期間中は温風で乾かせる。

## 共用施設

- 各階ロビー
- 1階多目的ホール
- テニスコート
- 体育館

## 生活

寮と秋田キャンパスの間では無料のシャトルバスを運行しており、授業やサークル等に合わせて利用できる。

門限は23時30分。年末年始の約1週間は閉寮期間で、在寮できない。学生生活の援助のため、寮監6名が交代で常駐している。

## Q&A 入寮

以下は「Q&A 入寮」に関する一覧である。

| 質問 | 回答 |
|---|---|
| 入寮希望者の倍率はどのくらいか。希望すれば必ず入寮できるか。 | 倍率は年度により異なるため回答できない。募集枠を超えた場合は優先基準の適用および抽選で選考する。 |
| 入寮前に寮内を見学できるか。 | 可能。大潟キャンパスに相談する。 |
| 入寮条件に保証人の収入上限はあるか。 | ない。 |
| 引越作業のため家族が寮内・居室に入れるか。 | 可能。寮到着時に寮監へ申し出る。 |
| 引越作業後に家族が部屋に泊まれるか。 | できない。近くの宿泊施設等を利用する。 |
| 台車やエレベーターはあるか。 | 台車は数台、エレベーターは2基ある。時間帯により混雑する。 |
| 入寮日の前に本人宛の荷物を発送できるか。 | できない。入寮日以降に到着するよう発送する。在学生は寮費発生を了承すれば入寮日前の荷物受入が可能。 |
| 入寮日を変更できるか。 | 入居する部屋の退去状況次第で可能。大潟キャンパスに相談する。 |
| 部屋番号を事前に知ることはできるか。 | できない。到着後に知らせる。荷物発送時は「住所+清新寮+氏名」と記載する。 |
| 入寮日から食事は提供されるか。 | 可能。入寮許可後、所定の手続により食事提供開始日を決定する。 |

## Q&A 寮費

以下は「Q&A 寮費」に関する一覧である。

| 質問 | 回答 |
|---|---|
| 振替口座に指定はあるか。 | 秋田銀行の本人名義口座を指定。ない場合は口座作成が必要。 |
| 入寮時の振替はいつから、いくら発生するか。 | 寮費は前納。入寮した月の翌月の振替日に、入寮月、翌月、翌々月の3か月分を振り替える。 |
| 残高不足等で振替不能の場合はどうなるか。 | 速やかに保証人へ通知し、通知記載の振込口座へ振り込む。 |
